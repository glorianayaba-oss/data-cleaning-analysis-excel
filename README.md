# Data Cleaning, Analysis & Visualization Project (Excel)

## Project Overview
This project focuses on cleaning, analyzing, and visualizing raw business data using Microsoft Excel to generate actionable insights and support data-driven decision-making.

## Problem Statement
The dataset contained inconsistencies, duplicates, and unstructured entries, making analysis difficult. The objective was to clean the data and present insights in a clear and visual format.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Excel Functions (IF, COUNTIF, VLOOKUP/XLOOKUP)
- Charts & Dashboards

## Process
1. Inspected the dataset to identify errors, duplicates, and missing values  
2. Cleaned and standardized the data for accuracy and consistency  
3. Analyzed data using Pivot Tables and Excel formulas  
4. Created charts and dashboards to visualize trends and insights  
5. Summarized findings into a clear report

## Key Insights
- Identified trends and patterns within the dataset  
- Improved data accuracy and reliability  
- Presented insights using clear visualizations for easy understanding  

## Outcome
This project demonstrates my ability to clean data, analyze datasets, and communicate insights through effective data visualization using Excel.
